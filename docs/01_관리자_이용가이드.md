## 1. 로그인 & 접속

1. https://onlinemeeting-zeta.vercel.app 접속
2. **Google 계정으로 로그인** 클릭
3. 관리자 권한이 부여된 Google 계정으로 로그인
4. 로그인 완료 시 **관리자 패널**로 자동 이동

## 2. 화면 구성

관리자 패널은 상단 탭 2개 그룹으로 구성됩니다.

| 그룹 | 탭 | 용도 |
|------|----|------|
| 설정 | 🗂 개원 단계 | 개원 단계 카테고리 추가/수정/삭제 |
| 설정 | 🏢 일반 업체 | 일반 제휴 벤더사 관리 + 계정 연결 |
| 설정 | 👨‍⚕️ 원장 관리 | 원장 권한 설정 (일사천리 / 비딩) |
| 설정 | 🏆 비딩 업체 | 비딩 전용 벤더사 관리 |
| 설정 | 📐 비딩 배정 | 원장별 비딩 업체 회차 배정 |
| 미팅 현황 | 📋 일반 미팅 | 전체 미팅 요청 현황 모니터링 |
| 미팅 현황 | ⚡ 일사천리 | 일사천리 미팅 현황 |
| 미팅 현황 | 🏆 비딩 현황 | 비딩 이벤트 및 슬롯 현황 |

## 3. 개원 단계 관리

업체를 분류하는 개원 단계(카테고리)를 관리합니다.

### 단계 추가

1. **🗂 개원 단계** 탭 클릭
2. **+ 단계 추가** 버튼 클릭
3. 아래 정보 입력:
   - **단계명** (필수): 예) 인테리어, 의료기기, 의료폐기물
   - **설명** (선택): 단계 설명
   - **순서**: 목록 표시 순서 (숫자)
   - **색상**: 14가지 컬러 중 선택
4. **저장** 클릭

### 단계 수정 / 삭제

- 각 단계 우측의 **수정** 또는 **삭제** 버튼 클릭
- ⚠️ 삭제 시 해당 단계에 속한 업체가 영향을 받을 수 있습니다

## 4. 일반 업체 관리

원장님이 미팅 신청할 수 있는 일반 제휴 벤더사를 관리합니다.

### 업체 등록

1. **🏢 일반 업체** 탭 클릭
2. 좌측에서 단계 선택 후 **+ 업체 추가** 클릭
3. 아래 정보 입력:
   - **업체명** (필수)
   - **개원 단계** (필수): 해당 업체가 속한 카테고리
   - 담당자명, 이메일, 전화번호, 웹사이트
   - 업체 소개 (선택)
4. **저장** 클릭

### 계정 연결

업체 담당자가 시스템에 가입하면, 해당 계정을 업체에 연결해야 인박스가 활성화됩니다.

**가입 링크 안내**
- 화면 상단 **제휴업체 전용 가입 링크** 옆 **링크 복사** 버튼으로 링크 복사
- 업체 담당자에게 해당 링크를 공유하여 가입 유도

**계정 연결 방법**
1. 업체 목록에서 **🔗 계정 연결** 버튼 클릭
2. 드롭다운에서 가입한 계정 선택 또는 이메일 직접 입력
3. **연결하기** 클릭
4. 연결 완료 시 **🟢 {담당자명}** 으로 표시됨

**계정 연결 해제**
- 연결된 업체의 **해제** 버튼 클릭

## 5. 원장 관리

가입한 원장님의 추가 기능 권한을 관리합니다.

1. **👨‍⚕️ 원장 관리** 탭 클릭
2. 원장 목록 확인 (이름 / 병원명 / 이메일 / 전공)
3. 각 원장의 권한 토글:
   - **⚡ 일사천리**: ON/OFF — 일사천리 탭 및 기능 활성화
   - **🏆 비딩**: ON/OFF — 비딩 탭 및 기능 활성화

> 검색창에 이름, 이메일, 병원명을 입력해 원장을 빠르게 찾을 수 있습니다.

## 6. 비딩 업체 관리

일반 업체와 분리된 **인테리어 비딩 전용** 벤더사를 관리합니다.

> 비딩 업체는 원장의 비딩 이벤트에 배정되어 슬롯을 선점하는 방식으로 참여합니다.

### 업체 등록

1. **🏆 비딩 업체** 탭 클릭
2. **+ 업체 추가** 클릭
3. 업체명, 담당자명, 이메일, 전화번호, 웹사이트, 업체 소개 입력
4. **활성화** 체크박스 확인 (체크 해제 시 비딩 배정에서 제외됨)
5. **저장** 클릭

### 계정 연결

일반 업체와 동일한 방식으로 계정을 연결합니다.
(가입 링크 공유 → 가입 완료 → **🔗 계정 연결** 버튼으로 연결)

---

## 7. 비딩 배정

비딩 권한이 있는 원장님별로 비딩 업체를 회차별로 배정합니다.

1. **📐 비딩 배정** 탭 클릭
2. 좌측 원장 목록에서 원장 선택
   - 목록에는 비딩 권한(🏆)이 활성화된 원장만 표시됩니다
3. 우측에서 회차 탭 선택: **1차** | **2차** | **3차**
4. 배정할 업체의 체크박스 클릭 (체크 = 배정, 해제 = 배정 취소)
5. 변경 즉시 저장됩니다

> 배정된 업체는 해당 원장의 비딩 이벤트가 열리면 슬롯을 선점할 수 있게 됩니다.

## 8. 미팅 현황 모니터링

### 📋 일반 미팅 / ⚡ 일사천리

전체 미팅 요청 현황을 확인합니다.

**상단 통계 카드**
| 항목 | 설명 |
|------|------|
| 총 미팅 요청 | 전체 미팅 수 |
| 응답 대기 | 업체가 아직 응답하지 않은 미팅 |
| 확정 완료 | 미팅 시간이 확정된 건 |
| 거절 | 업체가 거절한 건 |

**필터 및 검색**
- 상단 탭으로 상태별 필터링 (전체 / 응답 대기 / 확정 / 거절)
- 검색창: 원장명, 병원명, 업체명, 단계명으로 검색

**테이블 정보**
- 상태 뱃지 / 신청일 / 개원 단계 / 원장님 / 업체 / 확정 일시
- 확정된 미팅: Google Meet 링크 확인 가능
- 거절된 미팅: 거절 사유 확인 가능

**우측 미니 캘린더**: 확정된 미팅 날짜를 달력으로 확인

### 🏆 비딩 현황

**상단 통계 카드**
| 항목 | 설명 |
|------|------|
| 총 비딩 이벤트 | 전체 이벤트 수 |
| 진행 중 | 현재 슬롯 선점 가능한 이벤트 |
| 전체 슬롯 | 모든 이벤트의 슬롯 합계 |
| 선점 완료 | 업체가 선점한 슬롯 수 |

**필터**
- 회차 필터: 전체 / 1차 / 2차 / 3차
- 상태 필터: 전체 / 진행 중 / 마감 / 완료

**이벤트 카드 클릭** → 슬롯별 상세 현황 확인
- 슬롯 상태: **선점 완료** (초록) / **대기 중** (주황) / **취소** (회색)
- 선점한 업체명 확인 가능 (관리자만)

## 9. Google 캘린더 연동 관리

화면 우측 상단에 캘린더 연동 상태가 표시됩니다.

| 상태 | 의미 |
|------|------|
| **활성화됨** | 미팅 확정 시 Google Calendar에 자동 일정 생성 |
| **비활성화됨** | 자동 일정 생성 안 함 (Meet 링크는 항상 생성) |

- **토글 버튼**: 클릭으로 활성화/비활성화 전환
- **🔍 진단 버튼**: Google Calendar & Meet 연동 상태 점검
  - 환경변수 확인
  - Meet 스페이스 생성 테스트
  - Calendar 이벤트 생성/삭제 테스트

> ⚠️ 일사천리 미팅은 캘린더 토글과 무관하게 **항상** 전용 캘린더에 일정이 생성됩니다.
