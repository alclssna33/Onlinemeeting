# 벤더사 비딩 이용가이드

> **개비공 온라인 미팅 시스템** — 벤더사(업체) 비딩용
> 접속 URL: https://onlinemeeting-zeta.vercel.app

---

## 비딩이란?

비딩은 **선착순** 방식의 미팅 슬롯 선점 시스템입니다.

원장님이 미팅 가능한 시간 슬롯을 여러 개 올려두면, 배정된 업체들이 원하는 슬롯을 먼저 선점합니다. 다른 업체가 어떤 슬롯을 선점했는지는 **알 수 없습니다** (블라인드 방식).

> 비딩 기능은 **비딩 전용 업체**로 등록되고 관리자가 배정을 완료한 경우에만 이용할 수 있습니다.

## 1. 비딩 보드 진입

1. 로그인 후 인박스 화면 오른쪽 상단의 **비딩 보드** 버튼 클릭
   또는 직접 URL: https://onlinemeeting-zeta.vercel.app/vendor/bidding
2. 배정된 비딩 이벤트 목록이 표시됩니다

> 배정된 이벤트가 없으면 "배정된 비딩 이벤트가 없습니다. 관리자가 비딩 배정을 완료하면 여기에 표시됩니다." 메시지가 표시됩니다.

## 2. 비딩 보드 화면 구성

### 안내 배너

> ⚡ **선착순** 방식입니다. 원하는 시간대를 빠르게 선점하세요.
> 다른 업체가 선점한 슬롯은 "선점됨"으로 표시되며, 누가 선점했는지는 알 수 없습니다.

### 회차 필터

상단 탭으로 원하는 회차만 조회할 수 있습니다:
- **전체 회차** | **1차** | **2차** | **3차**

### 이벤트 카드

배정된 각 원장님의 비딩 이벤트가 카드로 표시됩니다.

| 항목 | 내용 |
|------|------|
| 회차 뱃지 | **1차 비딩** / **2차 비딩** / **3차 비딩** |
| 원장님 정보 | 원장명 + 병원명 |
| 이벤트 제목 | (원장님이 입력한 경우 표시) |
| 내 선점 현황 | **내 선점 {X}개** (선점한 경우) 또는 **미선점** (회색) |
| 슬롯 수 | 전체 슬롯 개수 |

### 우측 미니 캘린더

내가 선점한 슬롯의 날짜가 달력에 표시됩니다.

## 3. 슬롯 선점하기

1. 이벤트 카드 클릭 (▼) → 슬롯 목록 전개
2. 원장님 전달사항 확인 (있을 경우)
3. 원하는 시간대의 슬롯 확인:
   - **선점 가능** (초록 배지): 선점 가능한 슬롯
   - **선점됨** (회색 배지): 다른 업체가 이미 선점
   - **취소** (회색 배지): 이용 불가
4. **선점하기** 버튼 클릭 (회차 색상 배경)
5. 완료 시 "슬롯을 선점했습니다! 🎉" 토스트 알림 표시

> **주의**: 이벤트 1개당 슬롯 1개만 선점할 수 있습니다.
> 이미 선점한 이벤트의 다른 슬롯에는 **선점하기** 버튼이 비활성화됩니다.

### 선점 완료 후

- 슬롯 상태: **✓ 내 선점** (회차 색상 배경)으로 변경
- **Google Meet 링크** 자동 생성
- **Google Calendar** 일정 자동 등록
- 원장님에게 선점 알림 이메일 자동 발송

---

## 4. 선점 취소하기

내가 선점한 슬롯을 취소할 수 있습니다.

1. 해당 슬롯의 **선점 취소** 버튼 클릭 (빨간 배경)
2. 확인 후 취소 처리
3. "선점이 취소되었습니다." 토스트 알림 표시
4. 슬롯 상태가 **선점 가능**으로 돌아가고, 다른 업체가 선점할 수 있게 됩니다

> ⚠️ 취소하면 해당 슬롯의 Google Meet 링크도 사라집니다.

---

## 5. Google Meet 확인

선점한 슬롯에서 📹 버튼 클릭 → Google Meet 화상회의 입장

## 6. 슬롯 상태 안내

| 상태 | 표시 | 의미 |
|------|------|------|
| **선점 가능** | 초록 배지 + **선점하기** 버튼 | 내가 선점 가능 |
| **✓ 내 선점** | 회차 색상 배경 | 내가 선점 완료 |
| **선점됨** | 회색 배지 (버튼 없음) | 다른 업체가 먼저 선점 |
| **취소** | 회색 배지 | 이용 불가 슬롯 |
| 취소선 텍스트 | 회색 날짜 | 이미 지난 시간 (선점 불가) |

### 회차별 색상

| 회차 | 색상 |
|------|------|
| **1차** | 초록 |
| **2차** | 파란 |
| **3차** | 주황 |
