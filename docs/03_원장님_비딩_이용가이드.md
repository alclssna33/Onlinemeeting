# 원장님 비딩 이용가이드

> **개비공 온라인 미팅 시스템** — 원장님 비딩용
> 접속 URL: https://onlinemeeting-zeta.vercel.app

---

## 비딩이란?

비딩은 여러 업체가 **선착순으로 미팅 슬롯을 선점**하는 방식입니다.

원장님이 미팅 가능한 시간 슬롯을 여러 개 등록하면, 관리자가 배정한 업체들이 원하는 슬롯을 먼저 선점합니다. 업체 입장에서는 블라인드 경쟁 방식이라 서로 누가 선점했는지 알 수 없습니다.

> **비딩 탭은 관리자가 권한을 부여한 원장님에게만 표시됩니다.**

## 1. 비딩 화면 진입

1. 대시보드 상단 **🏆 비딩** 탭 클릭
2. 비딩 관리 화면으로 이동 (또는 **← 대시보드** 버튼으로 돌아갈 수 있음)

---

## 2. 비딩 이벤트 등록

### 이벤트 등록 순서

1. 화면 우측 상단 **+ 이벤트 등록** 버튼 클릭 (주황색)
2. 이벤트 등록 폼이 열립니다

### 입력 항목

#### 비딩 회차 선택
- **1차 비딩** | **2차 비딩** | **3차 비딩** 중 선택

#### 제목 (선택)
- 예: "인테리어 업체 비딩 1차"

#### 업체 전달사항 (선택)
- 슬롯을 선점할 업체들에게 전달할 내용
- 예: "미팅 전 포트폴리오 준비 요청", "원내 방문 가능 여부 확인 필요"

#### 미팅 슬롯 추가
- **+ 슬롯 추가** 버튼 클릭 → 날짜와 시간 선택
- 최소 1개 이상, 여러 개 추가 가능
- 각 슬롯 우측 **✕** 버튼으로 삭제 가능

3. **이벤트 등록** 버튼 클릭
4. 등록 완료 시 이벤트 목록에 **진행 중** 상태로 표시됨

> 이벤트를 등록하면 관리자가 배정한 업체들이 슬롯을 선점할 수 있게 됩니다.

---

## 3. 이벤트 상태 관리

각 이벤트 카드에서 상태를 변경할 수 있습니다.

| 상태 | 의미 | 가능한 액션 |
|------|------|------------|
| **진행 중** | 업체가 슬롯 선점 가능한 상태 | **마감** 또는 **완료** 버튼 |
| **마감** | 슬롯 선점 종료 (신규 선점 불가) | — |
| **완료** | 이벤트 종료 | — |

- **마감** 버튼: 더 이상 슬롯 선점을 받지 않을 때 클릭
- **완료** 버튼: 비딩이 완전히 끝났을 때 클릭

---

## 4. 슬롯 선점 현황 확인

이벤트 카드를 클릭하면 슬롯별 상세 현황을 볼 수 있습니다.

### 이벤트 카드 정보

- **{X}차 비딩** 뱃지 + 상태 뱃지
- 이벤트 제목 또는 생성일
- 전달사항 (있을 경우)
- **슬롯 진행률**: X / Y 슬롯 선점 (진행률 바)

### 슬롯 상세 정보

카드 클릭(▼) 시 슬롯 목록 전개:

| 항목 | 내용 |
|------|------|
| 번호 | 슬롯 순서 |
| 일시 | 예) "3월 10일 월요일 14:00" |
| 상태 | **선점 완료** (초록) / **대기 중** (주황) / **취소** (회색) |
| 선점 업체 | 어느 업체가 선점했는지 확인 가능 |
| Meet 링크 | 선점 완료 시 📹 버튼으로 Meet 링크 확인 |

### 우측 미니 캘린더
- 선점이 완료된 슬롯의 날짜가 달력에 표시됩니다

---

## 5. 비딩 회차 안내

비딩은 최대 3차까지 진행할 수 있습니다.

| 회차 | 색상 | 의미 |
|------|------|------|
| **1차** | 초록 | 첫 번째 비딩 |
| **2차** | 파란 | 두 번째 비딩 |
| **3차** | 주황 | 세 번째 비딩 |

- 상단 **전체 / 1차 / 2차 / 3차** 필터로 회차별 이벤트 조회 가능
- 같은 회차에 이벤트를 여러 개 등록할 수 있습니다
