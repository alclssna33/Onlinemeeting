## 1. 처음 가입하기

> 일반 로그인 페이지가 아닌 **벤더사 전용 가입 링크**를 통해 가입해야 합니다.
> 관리자에게 가입 링크를 받아 진행해주세요.

1. 관리자에게 받은 **벤더사 전용 가입 링크** 접속
2. **Google 계정으로 로그인** 클릭
3. 로그인 완료 시 자동으로 벤더사 권한이 부여됩니다
4. 이후 관리자가 시스템 내에서 귀사 업체 계정과 연결 처리

> 계정 연결이 완료되어야 미팅 요청 인박스를 이용할 수 있습니다.
> 연결 처리는 관리자에게 문의해주세요.

## 2. 인박스 화면 구성

로그인 후 **미팅 요청 인박스** 화면이 표시됩니다.

### 상단 요약

- **"안녕하세요, {업체명}님 · 총 {X}건"**
- **응답 필요 {X}건** (주황, 펄스 애니메이션): 아직 처리하지 않은 요청
- **확정 {X}건** (초록): 확정된 미팅 수

### 필터 & 검색 (좌측)

| 필터 | 설명 |
|------|------|
| **응답 필요** | 아직 수락/거절하지 않은 요청 |
| **확정됨** | 미팅 시간이 확정된 건 |
| **거절됨** | 내가 거절한 건 |
| **전체** | 전체 목록 |

- 검색창: 원장님 이름 또는 병원명으로 검색
- 단계 필터: 드롭다운으로 특정 개원 단계만 필터링
- 정렬: 최신 순 ↓ / 오래된 순 ↑

### 우측 미니 캘린더
- 확정된 미팅이 있는 날짜가 색상으로 표시됩니다

## 3. 미팅 요청 확인 & 처리

### 미팅 카드 확인

각 미팅 요청 카드에는 아래 정보가 표시됩니다:
- 원장님 성함 + 병원명
- 개원 단계 (예: 인테리어, 의료기기)
- 신청일
- 상태 뱃지

### 미팅 수락 (확정하기)

1. 미팅 카드의 **확인 · 처리** 버튼 클릭 → 카드 전개
2. **제안된 후보 시간** 목록 확인
   - 원장님이 제안한 희망 시간이 최대 5개까지 표시됩니다
3. 가능한 시간 **클릭하여 선택** (선택 시 컬러 배경으로 강조)
4. **✅ 확정하기** 버튼 클릭
5. 확정 완료 시:
   - Google Meet 링크 자동 생성
   - Google Calendar에 일정 자동 등록 (관리자 설정에 따라)
   - 원장님에게 확정 이메일 자동 발송 (Meet 링크 포함)

### 미팅 거절하기

1. 카드 전개 후 **거절** 버튼 클릭
2. 거절 사유 입력 (선택사항)
   - 예: "해당 시간대 일정이 꽉 찼습니다", "담당 지역이 아닙니다"
3. **거절 확정** 버튼 클릭

> 거절 사유는 원장님에게 표시됩니다. 작성해두시면 원장님이 재신청 시 참고할 수 있습니다.

---

## 4. 미팅 확정 후 안내

미팅이 확정되면 다음과 같이 처리됩니다:

- 미팅 카드 상태가 **확정됨** (초록)으로 변경
- **📅 확정된 날짜/시간** 표시
- **🎥 Google Meet 입장하기** 버튼 활성화
- 원장님 대시보드에 실시간으로 확정 알림 표시

---

## 5. 선정 & 탈락 결과 확인

같은 개원 단계에서 여러 업체가 미팅을 진행한 경우, 원장님이 최종 업체를 선정합니다.

### 최종 선정된 경우

화면 상단에 **초록색 배너** 표시:

> 🎉 **최종 선정 업체입니다!**
> {원장명} 원장님께서 귀사를 [{단계명}] 단계의 최종 파트너로 선정하셨습니다.
> 📞 원장님 연락처: {전화번호}

- 원장님 연락처가 공개됩니다 → 직접 연락하여 다음 단계를 진행하세요

### 탈락된 경우

미팅 카드 또는 상단 배너에 안내 표시:

> 📋 이 미팅에서 다른 업체가 최종 선정되었습니다.

---

## 6. Google Meet 입장하기

1. **확정됨** 필터 탭 클릭
2. 해당 미팅 카드에서 **🎥 Google Meet 입장** 버튼 클릭
3. 새 탭에서 Google Meet 화상회의 화면으로 이동

> "Meet 링크 준비 중"으로 표시될 경우 잠시 후 새로고침해주세요.

---

## 7. 상태 뱃지 안내

| 뱃지 | 색상 | 의미 |
|------|------|------|
| **응답 필요** | 주황 | 원장님 미팅 요청 — 수락 또는 거절 필요 |
| **확정됨** | 초록 | 미팅 시간 확정 완료 |
| **거절됨** | 회색 | 내가 거절한 미팅 |
| **🏆 최종 선정** | 초록 배너 | 해당 단계 최종 파트너로 선정됨 |
| **탈락** | 카드 내 안내 | 해당 미팅에서 다른 업체가 선정됨 |
