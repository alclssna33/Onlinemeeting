## 1. 처음 가입하기

1. https://onlinemeeting-zeta.vercel.app 접속
2. **Google 계정으로 로그인** 클릭
3. 최초 가입 시 프로필 설정 화면으로 이동됩니다
4. 아래 정보 입력 후 **시작하기** 클릭:
   - **닉네임 (원장님 성함)** (필수): 업체에 표시되는 이름
   - **전화번호** (필수): 업체 선정 시 해당 업체에 공개
   - **알림 수신 이메일** (선택): 미팅 알림을 받을 이메일. 비워두면 Gmail로 발송됨

> 닉네임과 전화번호는 이후 대시보드에서 언제든지 변경할 수 있습니다.

## 2. 대시보드 화면 구성

로그인 후 원장님 대시보드로 이동됩니다.

### 탭 구성

| 탭 | 설명 |
|----|------|
| 🏢 업체 찾기 | 개원 단계별 제휴 업체 검색 및 미팅 신청 |
| 📅 내 미팅 | 신청한 미팅의 상태 확인 및 관리 |
| 🏆 비딩 | 비딩 이벤트 관리 (관리자 권한 부여 시 표시) |
| ⚡ 일사천리 | 일사천리 미팅 목록 (관리자 권한 부여 시 표시) |

> 📅 내 미팅 탭의 숫자 뱃지는 응답 대기 중인 미팅 건수를 나타냅니다.

## 3. 업체 찾기 & 미팅 신청

### 업체 찾기

1. **🏢 업체 찾기** 탭 클릭
2. 좌측에서 **개원 단계** 선택 (예: 인테리어, 의료기기 등)
3. 우측에 해당 단계의 제휴 업체 카드 목록이 표시됨
4. 업체 카드에서 업체 정보 확인:
   - 업체명, 담당자, 업체 설명, 전화번호

### 미팅 신청

1. 원하는 업체 카드의 **미팅 신청** 버튼 클릭
2. 미팅 신청 모달이 열립니다

#### 희망 시간 선택
- **📅 날짜 · 시간 선택하기** 클릭 → 날짜와 시간 입력
- 최대 **5개**까지 희망 시간 추가 가능
- 후보 시간이 많을수록 확정 가능성이 높아집니다

#### 전달사항 입력 (선택)
- 업체에 미리 전달하고 싶은 내용 입력
- 예: "포트폴리오 준비 부탁드립니다", "병원 위치 안내 필요"

3. **미팅 신청하기 (X개 시간 제안)** 버튼 클릭
4. 신청 완료 메시지 확인 → **확인** 버튼

> 신청이 완료되면 업체 담당자에게 자동으로 이메일이 발송됩니다.

## 4. 내 미팅 확인

**📅 내 미팅** 탭에서 신청한 모든 미팅을 확인할 수 있습니다.

### 상단 통계

| 항목 | 설명 |
|------|------|
| 응답 대기 | 업체가 아직 응답하지 않은 미팅 |
| 확정 완료 | 업체가 시간을 확정한 미팅 |
| 거절/취소 | 업체가 거절하거나 취소된 미팅 |

### 섹션별 내용

#### ✅ 확정된 미팅
- 확정된 날짜/시간 표시 (📅)
- **🎥 Google Meet 입장하기** 버튼으로 화상 미팅 입장
- 해당 업체를 최종 선정하려면 **선정** 버튼 클릭

#### ⏳ 응답 대기 중
- 제안한 후보 시간 목록 표시
- 업체가 확정하면 자동으로 확정된 미팅으로 이동

#### 거절 / 취소
- 거절 사유 확인 가능
- **재신청** 버튼으로 다시 미팅 신청 가능

### 우측 미니 캘린더
- 확정된 미팅이 있는 날짜가 색상으로 표시됩니다
- 날짜에 마우스를 올리면 미팅 상세 정보 확인 가능

## 5. 미팅 확정 후 — 업체 선정하기

같은 개원 단계에서 여러 업체와 미팅을 진행한 후, 최종 1곳을 선정할 수 있습니다.

1. **업체 찾기** 탭 또는 **내 미팅 탭**에서 확정된 미팅 카드 확인
2. 최종 선정할 업체의 **선정** 버튼 클릭
3. 확인 후 처리됩니다:
   - 선정된 업체: **✅ 선정됨** 뱃지 표시
   - 나머지 업체: **탈락** 뱃지 표시

> **선정된 업체에게 자동으로 원장님 연락처(전화번호)가 공개됩니다.**
> 프로필에 전화번호가 등록되어 있어야 공개됩니다.

---

## 6. Google Meet 입장하기

1. **내 미팅** 탭 → **✅ 확정된 미팅** 섹션
2. 해당 미팅 카드의 **🎥 Google Meet 입장하기** 버튼 클릭
3. 새 탭에서 Google Meet 화상회의 화면으로 이동

> Meet 링크가 준비 중으로 표시될 경우: 업체 담당자가 미팅을 확정하는 과정에서 자동 생성됩니다. 잠시 후 새로고침해주세요.

## 7. 상태 뱃지 안내

업체 카드와 미팅 목록에 표시되는 상태 뱃지 의미입니다.

| 뱃지 | 의미 |
|------|------|
| **진행중** (주황) | 미팅 신청 후 업체 응답 대기 중 |
| **확정** (초록) | 업체가 미팅 시간을 확정함 |
| **거절** (회색) | 업체가 미팅을 거절함 |
| **✅ 선정됨** (초록) | 해당 업체를 최종 선정함 |
| **탈락** (회색) | 같은 단계에서 다른 업체가 선정됨 |
